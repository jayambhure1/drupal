# Drupal Site - My Claro Subtheme

A Drupal project setup with Composer-based installation and custom theme development.

## Project Structure

```
drupal/
├── drupalbycomposer/       # Main Drupal project with Composer
│   ├── web/                # Drupal core and modules
│   ├── vendor/             # Composer dependencies
│   ├── composer.json       # Composer dependencies file
│   └── db/                 # Database exports
└── README.md               # This file
```

For detailed setup and development information, see [drupalbycomposer/README.md](drupalbycomposer/README.md)
